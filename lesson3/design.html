Color Scheme

Primary Color: A lighter shade of blue that will be used for the navbar, footer, buttons, and the logo. This color will create a bold and modern look and will be the primary color that ties the website's design elements together.

#3498DB

Secondary Color: A soft gray tone that will be used for some text, buttons, and design features. This color will complement the primary color and will be used to create contrast and balance in the website's design.

#4B4B4B

Accents: A matte purple hue that will be used for design features such as icons, highlights, and other visual elements. This color will be used sparingly to draw the eye and create interest in the website's design.

#8C52FF

Text: Most of the text on the website will be in black to ensure readability and clarity. This will allow the primary and secondary colors to stand out and create visual interest without overwhelming the user.

Overall, this color scheme will create a modern, professional, and visually appealing website that is easy to navigate and use.
